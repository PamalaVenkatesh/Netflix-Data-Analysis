# Netflix-Data-Analysis-Movies and TV Shows available on Netflix across different countries. The data is arranged in a pivot-table format, where each row represents a country and the columns display the count of Movies and TV Shows available in that region. For example, countries like the United States and India have their respective movie and TV show counts listed in separate columns. The first two rows contain pivot table labels such as "Count of type" and "Column Labels," indicating that the sheet was generated from a summarized dataset. Overall, the file provides a clear comparison of Netflix content distribution across multiple countries.
